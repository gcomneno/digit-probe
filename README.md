# digit-probe
Analizzatore di sequenze numeriche (cifre o interi) con metriche:
chi-square, runs test, gaps, autocorrelazione, zlib ratio, n-gram, SchurProbe.

Il progetto è un vero strumento diagnostico per sequenze numeriche e bucket — già applicabile con successo al Turbo-Bucketizer (https://github.com/gcomneno/turbo-bucketizer).
